Hooks Reusable common attributes

About

Hooks are strings of text that can be reused throughout a theme or content on any one specific site. In other words, when working on a clients WordPress site, their address, and phone number can be implemented on the theme level, or in the content are using a snippet of PHP or shortcode. A shortcode is simply a word or phrase surrounded with brackets, like [gallery], when used in the content of a web site will pull special kind functionality, or simple content depending on the shortcode.

We are going to do a high level overview here, for more information check out our Wikipedia article.

Breadcrumbs

About

Breadcrumbs are auto-generated based on the page title in WordPress and can be used in multiple ways, using CSS to hide parents and the home button.

lexis_breadcrumbs('<');

This function allows you to change the delimiter, boundary or separator between the different parents and children.

If no delimiter is required place a non-breaking space tag, or a space instead.

CSS Output

CSS Class Translation
.current The current child page you are on has this class
.breadcrumb_home The home breadcrumb has this class
.breadcrumb_01 This is a sequence of nested parents, the number depends on the depth

PHP Snippet

<?php if (function_exists('lexis_breadcrumbs')) lexis_breadcrumbs('<'); ?>

Custom Fields Custom meta data

About

WordPress has Custom Fields which extends the capabilities for custom meta data on specific pages. This is a useful feature for page specific content, banners or imagery.

Parameters & valid values and descriptions

Parameters Values
$post_id (integer) The ID of the post from which you want the data. Use 1post->ID to get a post's ID (default)
$key (string) A string containing the name of the meta value you want. See our naming conventions, use meaningful keys like 'inner-banner'.
$single (boolean) If set to true then the function will return a single result, as a string. If false, or not set, then the function returns an array of the custom fields. This is not intuitive. For example, if you fetch a serialized array with this method you want $single to be true to actually get an unserialized array back. If you pass in false, or leave it out, you will have an array of one, and the value at index 0 will be the serialized string.

PHP Snippet

This snippet gets the post ID of the page, then looks for a custom field labeled 'custom-field'* and then prints that value. If there is no value, or no custom field set, the else statement should have a fall back.

* 'custom-field' is too arbitrary and should never be used. Describe what your content is.
<?php if ( get_post_meta($post->ID, 'custom-field', true) ) { 'custom-field' ?>
  <?php echo get_post_meta($post->ID, 'custom-field', true); 'custom-field' ?>
<?php } else { ?>
  <!-- back-up / default HTML goes here -->
<?php } ?>

Conditionals if/else statements

About

All of our sites have specific home pages, there are a few conditional statements that allows one to dynamically find out if one is on the homepage or another page of the web site. We'll give a few more conditions for examples, which are met or not and return true, or false.

Conditions

Condition Value
is_front_page() Displays true when the front of the site is displayed, whether it is posts or a page.
is_single() (string) When any single Post (or attachment, or custom Post Type) page is being displayed. (False for Pages)
is_single( '17' ) When Post 17 is being displayed as a single Post.
is_single( array( 17, 'beef-stew', 'Irish Stew' ) ) Returns true when the single post being displayed is either post ID 17, or the post_name is "beef-stew", or the post_title is "Irish Stew".
is_page() (string) When any Page is being displayed.
is_page( 42, 'about-me', 'About Me And Joe' ) Returns true when the Pages displayed is either post ID = 42, or post_name is "about-me", or post_title is "About Me And Joe".

PHP Snippet

This is an pretty standard if/else statement. Line 1 it's saying (if) this is the home page, and opens an area between line 1 and line 2 to do a specific set of HTML/CSS/PHP; on Line 2 says if this is not the home page, (else), display this HTML/CSS/PHP instead.

<?php if (is_front_page()) { ?>
<?php } else { ?>
<?php } ?>

Dynamic Navigations Using WordPress's Menu system

About

WordPress has a built in Menu system within the CMS which we take advantage of. This allows people to create, move, add to existing Menu's.

Most Common Parameters & Values

Parameters Values
theme_location (string) (optional) the location in the theme to be used--must be registered with register_nav_menu() in order to be selectable by the user
container (string) (optional) Whether to wrap the ul, and what to wrap it with. Allowed tags are div and nav. Use false for no container e.g. container => false
items_wrap (string) (optional) Whatever to wrap the items with an ul, and how to wrap them with

PHP Snippets

There are three registered Menu's in all of our sites. Therefore here are the three snippets for each.

Main Navigation
<?php
  wp_nav_menu(
    array(
      'theme_location' => 'main-nav',
      'container' => 'ul'
    )
  );
?>
Footer Navigation
<?php
  wp_nav_menu(
    array(
      'theme_location' => 'footer-nav',
      'container' => 'ul'
    )
  );
?>
Areas of Practice Navigation
<?php
  wp_nav_menu(
    array(
      'theme_location' => 'aop-nav',
      'container' => 'ul'
    )
  );
?>

Random Images snippet for random image banners

About

This snippet will make for a random image from a set of images.

Naming & Theory

Make sure you follow our naming conventions. This snippet is an example for the home page, so if you are using it on the an inner page change the alt tags. Also be sure to change the file name from home-banner- etc. to the proper name and file extension. This randomizes a set of numbers. So if you have rand(1,4) it will randomize 01, 02, 03, 04. Make sure you go two numbers deep on your naming for banner images just in case later on down the road the client adds banners we're all set.

Most Common Parameters & Values

Parameters Values
rand() (1,4) use this for the number of images you have, it's he range.

There are three registered Menu's in all of our sites. Therefore here are the three snippets for each.

Main Navigation
