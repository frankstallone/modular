Frameworks Also known as Libraries

What is a Framework?

JavaScript is considered by most the language of the web. It is object-oriented, meaning it uses objects which can contain data and methods that act upon that data. As the demands for JavaScript have been increasing and easier means to develop these dynamic interfaces was needed. Thus libraries, or frameworks as we'll call them here were created. For sake of keeping this simple, we use jQuery library (or framework). This can be found in /js/libs/jquery.js or relatively file name.

jQuery

From jQuery's website,

jQuery is a fast and concise JavaScript Library that simplifies HTML document traversing, event handling, animating, and Ajax interactions for rapid web development. jQuery is designed to change the way that you write JavaScript.

Since we are using jQuery we are able to code once, and know it will work cross-browser, and cross-platform. We also have at our disposal generous plugin's and scripts that have been written by brilliant developers at our disposal for getting things done!

Plugins One file, all your plugins organized.

What are plugins?

Plugins.js is meant to hold all of a site's plugin code. Instead of linking to many different files, we can improve performance by including plugin code directly in this one file. This works effectively for our use because we're creating relatively small websites for our clients. If we were doing larger projects like Lawyers.com where there were more than 50 pages, it would make more sense to modularize our scripts so that we do not bloat every page with code. Since we are really only going to be using 1 to 5 plugins max this is the best balance between performance and organization.

What are considered plugins?

Plugins are JavaScript code pre-written, tested, and standardized. For example, some JavaScript written for making a universal modal window that works cross browser

Scripts Calling that plugin? Call toll free here.

What are scripts?

Scripts.js is meant to hold all of the site-specific javascript. For instance, if we are using SlidesJS on a site, we have that plugin's code in the plugins.js file and we'll write our specific calls for that plugin in our scripts.js. This works because we call the scripts.js in our HTML after the plugins.js.

Rules of the road

Comment your code. Make sure you add your name to the author list if you are going to be making edits here, that way if someone comes back they know who to ask if they need ot figure out what does what, or to fix an issue that my have occured.

SlidesJS a simple slideshow plugin for jQuery.

About

SlidsJS is the simple slideshow plugin for jQuery we use. There are many useful features you can read more about here.

Minified Plugin

The minified code should be in the client folder automatically. In other words /clientfolder/!base-build/js/plugins.js. Take a look and see if the glossary in that plugins.js file includes SlidesJS, if not download the minified source here.

Common Used Options

This is a snippet for reference for the scripts.js with options already set. You can find more options in the docs section of the SlidesJS website. The following snippet looks for containing HTML element with the class of .banner.

$(function(){
	$(".banner-home").slides({
		preload: true,
		play: 7000,
		pause: 2500,
		effect: 'fade',
		crossfade: true,
		container: 'home-banner-container',
		pagination: false,
		generatePagination: false,
		randomize: true
	});
});
# Option Description
3 play 7000 milliseconds, and accepts any positive number that will set to true. Default is 0 or false.
4 pause 2500 milliseconds, will pause the slideshow on next/prev or pagination click. Default is 0 or false.
5 effect 'fade', also accepts 'slide' or 'slide, fade'. Default is 'slide'.
6 crossfade true, crossfade images in a image based slideshow. Default value is false.
7 container 'home-banner-container', class name for slides container. Default value is 'slides_container'.
8 pagination false, if you're using pagination you can set to true.
9 generatePagination false, Auto generate pagination. Default value is 'true'.
10 randomize true, set false to play slides in order.

The above is just an example!

The above is a snippet of code a developer uses to start their coding. Below is the minimal code required to get SlidesJS working. 
$(function(){
    $("#slides").slides();
});
